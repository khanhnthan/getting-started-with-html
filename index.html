Part 2: HTML

HTML is made up of:
plain text = text that is unformated
text content = what you see
markup = what it looks like
references to other documents, e.g. imges & videos
links to other pages

HTML markup is made of tags. Tags usually have an opening tag, <tag name>, content, and a closing tag, </tag name>. Some tags like the break tag, <br>, does not have a closing tag and are called void tags. Complete tags can also be referred to as elements. Tags may also have attributes such as src for the image tag. An example of a tag: <b>This is a bold tag.</b>

There are two basic types of tags: inline elements and block elements.

INLINE ELEMENTS:

<br>makes a line break </br>
<em>italics </em>
<strong>bold </strong>
<b>also does bold </b>
<img src="link to image" alt="text in case image doesn't work>
<a href="url">creates an html link</a>
<span>content </span>

BLOCK ELEMENTS:

<p>creates a paragraph </p>
<div>more complex than a paragraph </div>
<form>content </form>

HTML treats all continuous white space as a single white space.
